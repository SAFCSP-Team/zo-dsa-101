# Recursion

Recursion is a powerful technique used to solve complex problems by breaking them into smaller, simpler versions of the same problem.

## Concept

A recursive algorithm is one where a function **calls itself** within its own definition. Each call works on a smaller version of the problem until a stopping condition is reached.

Every recursive function must have two parts:

1. **Base case**: The condition that stops the recursion.
2. **Recursive call**: The statement where the function calls itself with a smaller input.

Recursion can be:

- **Single**: The function calls itself once per invocation.
- **Multiple**: The function calls itself more than once per invocation (e.g. tree traversal).

>[!NOTE]
> Without a base case, the function will call itself indefinitely and cause a stack overflow error.

## Example

### Factorial

The factorial of a non-negative integer N is written as N! and defined as:

```
0! = 1
N! = N x (N-1)!
```

For example, calculating 3!:

```
3! = 3 x 2!
   = 3 x 2 x 1!
   = 3 x 2 x 1 x 0!
   = 3 x 2 x 1 x 1
   = 6
```

### Implementation

```java
public class Recursion {

    public static int factorial(int n) {
        // Base case: 0! = 1
        if (n == 0) {
            return 1;
        }
        // Recursive call: n! = n * (n-1)!
        return n * factorial(n - 1);
    }

    public static void main(String[] args) {
        System.out.println(factorial(3));
    }
}
```

**Output:**
```
6
```

### Tree Traversal

Recursion is a natural fit for tree traversal. Below is the same pre-order print operation implemented first with a stack, then with recursion.

**Using a stack:**

```java
public void print() {
    if (root == null) {
        System.out.println("Tree is empty");
        return;
    }

    Stack<Node> stack = new Stack<>();
    stack.push(root);

    while (!stack.isEmpty()) {
        Node current = stack.pop();
        System.out.print(current.data + " ");

        if (current.right != null) stack.push(current.right);
        if (current.left != null) stack.push(current.left);
    }
}
```

**Using recursion:**

```java
public void print(Node node) {
    // Base case
    if (node == null) return;

    // Print current node
    System.out.print(node.data + " ");

    // Recursive call on left subtree
    print(node.left);

    // Recursive call on right subtree
    print(node.right);
}
```

Call it from `main` by passing the root:

```java
public static void main(String[] args) {
    BinaryTree tree = new BinaryTree(new Node(1));
    tree.addLeft(1, 2);
    tree.addRight(1, 3);
    tree.addLeft(2, 4);

    tree.print(tree.root);
}
```

**Output:**
```
1 2 4 3
```

>[!NOTE]
> Both implementations produce the same pre-order output. The recursive version is shorter because the call stack itself manages the traversal state, replacing the stack object.
