# Arrays

Imagine two libraries: one organized, one scattered. In the organized library, books are arranged by genre — each category in its own section. In the scattered one, books are randomly placed with no system. The difference is immediately felt: finding a specific book is effortless in one and frustrating in the other.

The same principle applies in programming. When we need to store a large number of similar values — like a list of weekdays, student names, or scores — using separate variables for each one is impractical. Instead, we use an **array**: a single, organized structure that holds all related elements together.

## Concept

An `Array` is a collection of similar data elements grouped together under one name.

- Arrays can store **primitive** types (integers, characters, booleans) or **non-primitive** types (objects).
- The `length` of an array is the **total number of elements** it contains.
- Each element has an `index` — a unique numeric position starting from **0**.
- If an array has size **n**, the last valid index is **n-1**.
- Any element can be accessed directly using its index.

![Array concept — char vowels with indices 0 to 4](./images/Introduction_to_Arrays-01.png)

>[!NOTE]
> In most programming languages, array indices start from 0 and increment by 1.



## Types of Arrays

Arrays are categorized based on their number of dimensions:

- **One-Dimensional (1D) Array**: Elements arranged in a single row — the simplest and most common form.
- **Multi-Dimensional Array**: Has more than one dimension (rows and columns, or layers). The most common is the two-dimensional array.

![Array types — 1D, 2D, and 3D](./images/Introduction_to_Arrays-03.png)



## One-Dimensional Arrays

A one-dimensional array is a linear sequence of elements stored under a single name.


### Examples

#### Primitive

Before creating a full array, let's start with a single element. For example, an array holding only the letter `'A'`:

![Single element array visualization](./images/Introduction_to_Arrays-02.png)

```java
// 1. Declare
char[] vowels;

// 2. Allocate
vowels = new char[1]; // Or combine both steps -> char[] vowels = new char[1];

// 3. Assign
vowels[0] = 'A';

// Update value
vowels[0] = 'a';
```

Extending this to the full 5-element vowel array — the only change is the size:

```java
char[] vowels = new char[5];
vowels[0] = 'a';
vowels[1] = 'e';
vowels[2] = 'i';
vowels[3] = 'o';
vowels[4] = 'u';
```

Or more concisely in one step:

```java
char[] vowels = {'a', 'e', 'i', 'o', 'u'};
```

Print an array element: 
```java
System.out.println(vowels[1]); // Output: e
```

Print the **length** of the array: 
```java
System.out.println(vowels.length); // Output: 5
```

To perform any operations on each element of an array, you can use loops. For example, printing all elements of the "vowels" array:

```java
for (int i = 0; i < vowels.length; i++) {
    System.out.println(vowels[i]);
}
```

#### Non-Primitive

Arrays can also hold non-primitive types (objects) such as the `Person` class:

```java
Person[] peopleArray;
```
Let's implement a `Person` array.

Create the `Person` class:

```java
class Person {
    private String name;
    private int age;

    public Person(String name, int age) {
        this.name = name;
        this.age = age;
    }

    @Override
    public String toString() {
        return "Name: " + name + " Age: " + age;
    }
}
```

Creating the array and printing some information:

```java
public static void main(String[] args) {
    Person[] peopleArray = new Person[3];
    peopleArray[0] = new Person("Alice", 25);
    peopleArray[1] = new Person("Bob", 30);
    peopleArray[2] = new Person("Charlie", 35);


    System.out.println("The count = " + peopleArray.length);
    System.out.println("The secondElement = " + peopleArray[1]);
}
```

**Output:**
```
The count = 3
The secondElement = Name: Bob Age: 30
```



## Multi-Dimensional Arrays

Multi-dimensional arrays organize data in more complex structures. They are useful for representing tables (2D), 3D graphics, and other layered data. Each additional dimension adds another level of organization.

- `Two-Dimensional Array`: Organizes elements in **rows and columns** — also called a matrix.
- `Three-Dimensional Array`: Adds a **layer** dimension on top of rows and columns.


### Length

The `length` property behaves differently depending on the number of dimensions:

- **1D:** `oneDArray.length` → total number of elements
- **2D:**
  - `twoDArray.length` → number of rows
  - `twoDArray[0].length` → number of columns in the first row
- **3D:**
  - `threeDArray.length` → number of layers
  - `threeDArray[0].length` → number of rows in the first layer
  - `threeDArray[0][0].length` → number of columns in the first layer's first row

### Accessing Elements

Provide one index per dimension:

- **2D:** `twoDArray[0][3]` → row 0, column 3
- **3D:** `threeDArray[0][0][2]` → layer 0, row 0, column 2

![Multi-dimensional array element access](./images/Multi_Dimensional_Arrays.jpg)

### Examples

#### Primitive

```java
public static void main(String[] args) {
    int[][] twoDArray = {
        {1, 2, 3},
        {4, 5, 6},
        {7, 8, 9}
    };

    int rows = twoDArray.length;
    int columns = twoDArray[0].length;
    System.out.println("Number of rows: " + rows);
    System.out.println("Number of columns: " + columns);

    int element = twoDArray[1][2]; // Row 1, Column 2
    System.out.println("Element at (1, 2): " + element);
}
```

**Output:**
```
Number of rows: 3
Number of columns: 3
Element at (1, 2): 6
```

#### Non-Primitive

>[!NOTE]
> The `Person` class is the same one defined in the One-Dimensional Arrays section above.

```java
public static void main(String[] args) {
    Person[][] twoDArray = new Person[3][3];
    twoDArray[0][0] = new Person("John", 25);
    twoDArray[0][1] = new Person("Alice", 30);
    twoDArray[0][2] = new Person("Bob", 35);
    twoDArray[1][0] = new Person("Sarah", 28);
    twoDArray[1][1] = new Person("Mike", 32);
    twoDArray[1][2] = new Person("Emily", 27);

    int rows = twoDArray.length;
    int columns = twoDArray[0].length;
    System.out.println("Number of rows: " + rows);
    System.out.println("Number of columns: " + columns);

    Person element = twoDArray[1][2];
    System.out.println(element);
}
```

**Output:**
```
Number of rows: 3
Number of columns: 3
Name: Emily Age: 27
```

>[!NOTE]
> When creating a 2D array in Java, specify both dimensions explicitly or provide the elements immediately.
