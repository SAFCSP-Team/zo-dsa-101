# Queue

Have you ever been in a line? We usually form a line whenever we want to be served, whether it's at a coffee shop or a bank. When we form a line, the serving process becomes much easier and controllable. Each new customer who comes in will join the end of the line, and the first customer in line will be served first. After a customer gets served, they leave the line.

The same concept is used in data structures to organize data efficiently; it is called a `queue`.

## Concept

A `queue` is a **linear data structure** where elements are stored and processed in order. The first element in the queue is called the `head` (or `front`), and the last element is called the `tail` (or `rear`).

A queue follows **"first in, first out" (FIFO)** behavior; the first element added is the first to be removed, just like a real line.

**Queue operations:**

- `enqueue(x)`: Add element x at the end of the queue.
- `dequeue()`: Remove and return the first element from the queue.
- `isEmpty()`: Return true if the queue is empty.
- `isFull()`: Return true if the queue is full.


>[!NOTE]
> Each programming language may use different names for these operations. For example: enqueue = `addLast` / `insert`, and dequeue =  `removeFirst` / `delete`.

**Why do we use a queue?**

To ensure that elements are processed in the order they were added, making it possible to manage tasks or requests in a systematic and predictable manner.

## Types of Queue

- **Linear Queue**: The most basic form, where elements are stored and processed in a linear sequential manner.
- **Circular Queue**: Arranges data in a circular order where the rear end is connected back to the front end, solving the wasted space problem of the linear queue.
- **Deque (Double-Ended Queue)**: Allows insertion and removal of elements from both ends.
- **Priority Queue**: Elements with higher priority are dequeued before elements with lower priority. If elements share the same priority, they are served in order.



## Linear Queue

A **linear queue** is the simplest type of queue, where elements are stored sequentially. The first element is the `head`, and the last is the `tail`.

**Operations:**

- **Enqueue**: Insert a node at the end of the queue at the `rear`/`tail`.
- **Dequeue**: Delete a node from the start of the queue at the `head`/`front`.

### Implementation

#### Using Array

1. Declare the class and attributes:

```java
class QueueArray {
    int array[];
    int front;
    int rear;
    int size = 5;
}
```

2. Add a constructor:

```java
class QueueArray {
    int array[];
    int front;
    int rear;
    int size = 5;

    public QueueArray() {
        array = new int[size];
        front = -1;
        rear = -1;
    }
}
```

3. Add `isEmpty()` and `isFull()` methods:

```java
boolean isEmpty() {
    return front == -1;
}

boolean isFull() {
    return (rear == size - 1);
}
```

4. Add `enqueue()` and `dequeue()` methods:

```java
void enqueue(int item) {
    if (isFull()) {
        System.out.println("Overflow");
        return;
    }
    if (isEmpty()) {
        front = 0;
        rear = 0;
    } else {
        rear++;
    }
    array[rear] = item;
}

void dequeue() {
    if (isEmpty()) {
        System.out.println("Queue is empty. Nothing to dequeue");
    } else if (front == rear) {
        front = rear = -1;
    } else {
        front++;
    }
}
```

5. Use the class:

```java
public class Main {
    public static void main(String[] args) {
        QueueArray queue = new QueueArray();

        queue.enqueue(10);
        queue.enqueue(20);
        queue.enqueue(30);
        queue.dequeue();
        queue.dequeue();
        queue.enqueue(40);
        queue.enqueue(50);

        System.out.println("Front: " + queue.front);
        System.out.println("Rear: " + queue.rear);
    }
}
```

**Output:**
```
Front: 2
Rear: 4
```

>[!NOTE]
> A limitation of the linear array queue is that even after dequeuing elements, the freed positions at the front cannot be reused — the queue is considered full once `rear == size - 1`. A **circular queue** solves this problem.

#### Using Linked List

1. Declare the `Node` and `QueueLinkedList` classes:

```java
class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
        this.next = null;
    }
}

class QueueLinkedList {
    private Node front;
    private Node rear;

    QueueLinkedList() {
        this.front = null;
        this.rear = null;
    }
}
```

2. Add `enqueue()`, `dequeue()`, and `display()` methods:

```java
public void enqueue(int item) {
    Node newNode = new Node(item);
    if (front == null) {
        front = newNode;
        rear = newNode;
        return;
    }
    rear.next = newNode;
    rear = newNode;
}

public int dequeue() {
    if (front == null) {
        System.out.println("Queue is empty!");
        return -1;
    }
    int result = front.data;
    front = front.next;

    // Update the rear if the queue is now empty
    if (front == null) {
        rear = null;
    }

    return result;
}

public void display() {
    Node current = front;
    while (current != null) {
        System.out.println(current.data);
        current = current.next;
    }
}
```

3. Use the class:

```java
public class Main {
    public static void main(String[] args) {
        QueueLinkedList queue = new QueueLinkedList();

        queue.enqueue(10);
        queue.enqueue(20);
        queue.enqueue(30);
        queue.dequeue();
        queue.enqueue(40);

        queue.display();
    }
}
```

**Output:**
```
20
30
40
```

```java
public class QueueArray {
    String array[];
    int front;
    int rear;
    final int size;

    QueueArray(int size) {
        this.size = size;
        this.front = -1;
        this.rear = -1;
        this.array = new String[size];
    }

    public boolean isEmpty() {
        return front == -1;
    }

    public boolean isFull() {
        return rear == size - 1;
    }

    public void enqueue(String element) {
        if (isFull()) {
            System.out.println("Queue is full");
            return;
        }
        if (isEmpty()) {
            front = rear = 0;
        } else {
            rear = rear + 1;
        }
        array[rear] = element;
    }

    public void dequeue() {
        if (isEmpty()) {
            System.out.println("Queue is empty!");
            return;
        }
        if (front == rear) {
            front = rear = -1;
        } else {
            front++;
        }
    }

    public void display() {
        for (int i = front; i <= rear; i++) {
            System.out.println(array[i]);
        }
    }

    public static void main(String[] args) {
        QueueArray que = new QueueArray(3);

        que.enqueue("Abdulmalik");
        que.enqueue("Khalid");
        que.enqueue("Fahad");

        que.dequeue();

        que.display();
    }
}
```

```java
class User {
    String username;
    String password;

    User(String username, String password) {
        this.username = username;
        this.password = password;
    }

    public String getUser() {
        return this.username + ", " + this.password;
    }

    @Override
    public String toString() {
        return username + ", " + password;
    }
}

public class QueueArray {
    final int SIZE = 5;
    User items[] = new User[SIZE];
    int front, rear;

    QueueArray() {
        front = -1;
        rear = -1;
    }

    public boolean isFull() {
        return rear == SIZE - 1;
    }

    public boolean isEmpty() {
        return (front == -1);
    }

    public void enQueue(User element) {
        if (isFull()) {
            System.out.println("Queue is full");
            return;
        }
        if (front == -1) {
            front = 0;
        }
        rear++;
        items[rear] = element;
    }

    public User deQueue() {
        if (isEmpty()) {
            System.out.println("Queue is empty");
            return null;
        }
        User element = items[front];
        if (front >= rear) {
            front = -1;
            rear = -1;
        } else {
            front++;
        }
        System.out.println(element + " Deleted");
        return element;
    }

    public void display() {
        if (isEmpty()) {
            System.out.println("Queue is empty!");
        } else {
            for (int i = front; i <= rear; i++)
                System.out.println("Item -> " + items[i].getUser());
        }
    }

    public static void main(String[] args) {
        QueueArray queueArray = new QueueArray();

        queueArray.enQueue(new User("fahad12", "P@ssw0rd"));
        queueArray.enQueue(new User("sara97", "123456"));
        queueArray.enQueue(new User("devahmed", "secret"));

        queueArray.display();
        queueArray.deQueue();
        queueArray.display();
    }
}
```

**Output:**
```

Item -> sara97, 123456
Item -> devahmed, secret
fahad12, P@ssw0rd Deleted
Item -> sara97, 123456
Item -> devahmed, secret
```


## Circular Queue

A **circular queue** solves the wasted space problem of a linear queue. Instead of considering the queue full when `rear == size - 1`, the rear wraps around to the beginning of the array using the modulo `%` operator, reusing freed positions.

### Implementation

The circular queue uses the same class structure as the linear queue. Only `isFull()`, `enqueue()`, and `dequeue()` are modified:

```java
class QueueArray {
    int array[];
    int front;
    int rear;
    int size = 5;

    public QueueArray() {
        array = new int[size];
        front = -1;
        rear = -1;
    }

    boolean isEmpty() {
        return front == -1;
    }

    boolean isFull() {
        return (front == 0 && rear == size - 1) || (front == (rear + 1) % size); // The queue is full when the next position of rear equals front.
    }

    void enqueue(int item) {
        if (isFull()) {
            System.out.println("Overflow");
            return;
        }
        if (front == -1) {
            front = 0;
            rear = 0;
        } else {
            rear = (rear + 1) % size; // wraps around
        }
        array[rear] = item;
    }

    void dequeue() {
        if (isEmpty()) {
            System.out.println("Queue is empty. Nothing to dequeue");
        } else if (front == rear) {
            front = rear = -1;
        } else {
            front = (front + 1) % size; // wraps around
        }
    }
}
```

**Use the class:**

```java
public class Main {
    public static void main(String[] args) {
        QueueArray queue = new QueueArray();

        queue.enqueue(10);
        queue.enqueue(20);
        queue.enqueue(30);
        queue.dequeue();
        queue.dequeue();
        queue.enqueue(40);
        queue.enqueue(50);

        System.out.println("Front: " + queue.front);
        System.out.println("Rear: " + queue.rear);
    }
}
```

**Output:**
```
Front: 2
Rear: 4
```



## Deque (Double-Ended Queue)

A **deque** (double-ended queue) is a type of queue that allows insertion and deletion from **both ends**.

<img width="1121" height="196" alt="Screenshot 2026-06-18 at 4 46 58 PM" src="https://github.com/user-attachments/assets/56e140e2-e47b-4dcb-abe4-8a33e597cc3f" />


**Operations:**

- `addFirst(x)`: add an element at the beginning of the queue.
- `addLast(x)`: add an element at the end of the queue.
- `deleteFirst()`: remove the first element.
- `deleteLast()`: remove the last element.


### Example

#### Primitive

```java
class DequeArray {
    int arr[];
    int front;
    int rear;
    int size;

    public DequeArray(int size) {
        arr = new int[size];
        front = -1;
        rear = 0;
        this.size = size;
    }

    boolean isFull() {
        return ((front == 0 && rear == size - 1) || front == rear + 1);
    }

    boolean isEmpty() {
        return (front == -1);
    }

    void addFront(int key) {
        if (isFull()) {
            System.out.println("Overflow");
            return;
        }
        if (front == -1) {
            front = 0;
            rear = 0;
        } else if (front == 0) {
            front = size - 1;
        } else {
            front = front - 1;
        }
        arr[front] = key;
    }

    void addRear(int key) {
        if (isFull()) {
            System.out.println("Overflow");
            return;
        }
        if (front == -1) {
            front = 0;
            rear = 0;
        } else if (rear == size - 1) {
            rear = 0;
        } else {
            rear = rear + 1;
        }
        arr[rear] = key;
    }

    void deleteFront() {
        if (isEmpty()) {
            System.out.println("Queue Underflow");
            return;
        }
        if (front == rear) {
            front = -1;
            rear = -1;
        } else if (front == size - 1) {
            front = 0;
        } else {
            front = front + 1;
        }
    }

    void deleteRear() {
        if (isEmpty()) {
            System.out.println("Underflow");
            return;
        }
        if (front == rear) {
            front = -1;
            rear = -1;
        } else if (rear == 0) {
            rear = size - 1;
        } else {
            rear = rear - 1;
        }
    }

    int getFront() {
        if (isEmpty()) {
            System.out.println("Underflow");
            return -1;
        }
        return arr[front];
    }

    int getRear() {
        if (isEmpty()) {
            System.out.println("Underflow");
            return -1;
        }
        return arr[rear];
    }

    public static void main(String[] args) {
        DequeArray deque = new DequeArray(5);

        deque.addFront(7);
        deque.addFront(6);
        deque.addFront(5);
        deque.addRear(8);
        deque.addRear(9);

        System.out.println("Front: " + deque.getFront());
        System.out.println("Rear: " + deque.getRear());
    }
}
```

**Output:**
```
Front: 5
Rear: 9
```
