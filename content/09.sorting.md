# Sorting Algorithms

Sorting is arranging items in a particular order so they can be accessed easily. We rely on it constantly. For example, when smartphones sort contact lists alphabetically, finding a specific name becomes fast and effortless instead of scanning a random list.

## Concept

`Sorting` is a data structure operation used to make searching and arranging elements easier. Sorted data improves search speed, enables more efficient algorithms, and generally boosts application performance.

There are several fundamental sorting algorithms, each with a different strategy:

- **Selection Sort**: repeatedly selects the smallest element from the unsorted part and moves it to the sorted part.
- **Bubble Sort**: repeatedly compares adjacent elements and swaps them if they're out of order.
- **Insertion Sort**: builds the sorted array one element at a time by inserting each into its correct position.
- **Merge Sort**: divides the list into halves, sorts each half, then merges them back together.
- **Quick Sort**: picks a pivot and partitions the list into elements smaller and larger than the pivot.

## Selection Sort

Selection sort divides the array into a **sorted** and **unsorted** section. It repeatedly finds the smallest element in the unsorted section and swaps it with the first unsorted position, growing the sorted section by one each time.

<img width="910" src="https://github.com/SAFCSP-Team/data-structures-and-algorithms-bootcamp/assets/148945652/00dea9ba-b28a-4d47-82a8-0ba4068672a3" />

**Example: sorting `[8, 3, 2, 9, 7]`**

| Step | Smallest found | Sorted section | Unsorted section |
|---|---|---|---|
| 1 | 2 | `2` | `3, 8, 9, 7` |
| 2 | 3 | `2, 3` | `8, 9, 7` |
| 3 | 7 | `2, 3, 7` | `8, 9` |
| 4 | 8 | `2, 3, 7, 8` | `9` |
| 5 | 9 | `2, 3, 7, 8, 9` | (empty) |

### Implementation

```java
public class SelectionSort {

    public static void selectionSort(int[] arr) {
        int n = arr.length;

        for (int i = 0; i < n - 1; i++) {
            int minIndex = i;

            // Find the index of the minimum element in the unsorted part
            for (int j = i + 1; j < n; j++) {
                if (arr[j] < arr[minIndex]) {
                    minIndex = j;
                }
            }

            // Swap the found minimum element with the first unsorted element
            int temp = arr[minIndex];
            arr[minIndex] = arr[i];
            arr[i] = temp;
        }
    }

    public static void printArray(int[] arr) {
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] numbers = {9, 2, 5, 1, 7, 4};

        System.out.print("Unsorted numbers: ");
        printArray(numbers);

        selectionSort(numbers);

        System.out.print("Sorted numbers: ");
        printArray(numbers);
    }
}
```

**Output:**
```
Unsorted numbers: 9 2 5 1 7 4 
Sorted numbers: 1 2 4 5 7 9 
```

## Bubble Sort

Bubble sort repeatedly compares each pair of adjacent elements and swaps them if they're out of order. With each full pass, the largest remaining element "bubbles" up to its correct position at the end.

<img width="910" src="https://github.com/SAFCSP-Team/data-structures-and-algorithms-bootcamp/assets/148945652/09c84453-6e78-4086-8676-d5152f73fcce" />

**Example: sorting heights `[160, 175, 155, 168]`**

| Pass | Comparisons | Result |
|---|---|---|
| 1 | 160 vs 175 (no swap), 175 vs 155 (swap), 175 vs 168 (swap) | `160, 155, 168, 175` |
| 2 | 160 vs 155 (swap), 160 vs 168 (no swap), 168 vs 175 (no swap) | `155, 160, 168, 175` |
| 3 | No swaps needed | `155, 160, 168, 175` (sorted) |

### Implementation

```java
public class BubbleSort {

    public static void bubbleSort(int[] arr) {
        int n = arr.length;

        for (int i = 0; i < n - 1; i++) {
            // Last i elements are already sorted
            for (int j = 0; j < n - i - 1; j++) {
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }

    public static void printArray(int[] arr) {
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] numbers = {9, 2, 5, 1, 7, 4};

        System.out.print("Unsorted numbers: ");
        printArray(numbers);

        bubbleSort(numbers);

        System.out.print("Sorted numbers: ");
        printArray(numbers);
    }
}
```

**Output:**
```
Unsorted numbers: 9 2 5 1 7 4 
Sorted numbers: 1 2 4 5 7 9 
```

## Insertion Sort

Insertion sort is similar to sorting playing cards in your hand. You take one card at a time and insert it into its correct position among the cards you're already holding, which remain sorted at all times.

<img width="910" src="https://github.com/SAFCSP-Team/data-structures-and-algorithms-bootcamp/assets/148945652/25900738-2532-4631-8076-1d82bf7d8bdc" />

**Example: sorting `[4, 3, 2, 10, 12, 1, 5, 6]`**

- Compare `3` with `4`. Since `3 < 4`, swap them, giving `[3, 4, 2, 10, 12, 1, 5, 6]`
- Compare `2` with `4`, then with `3`. Shift both, giving `[2, 3, 4, 10, 12, 1, 5, 6]`
- Continue this process until the entire array is sorted.

### Implementation

```java
public class InsertionSort {

    public static void insertionSort(int[] arr) {
        int n = arr.length;

        for (int i = 1; i < n; i++) {
            int key = arr[i];
            int j = i - 1;

            // Shift elements greater than key one position ahead
            while (j >= 0 && arr[j] > key) {
                arr[j + 1] = arr[j];
                j--;
            }

            arr[j + 1] = key;
        }
    }

    public static void printArray(int[] arr) {
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] numbers = {5, 2, 4, 6, 1, 3};

        System.out.print("Original Array: ");
        printArray(numbers);

        insertionSort(numbers);

        System.out.print("Sorted Array: ");
        printArray(numbers);
    }
}
```

**Output:**
```
Original Array: 5 2 4 6 1 3 
Sorted Array: 1 2 3 4 5 6 
```

## Merge Sort


Merge sort follows the **divide-and-conquer** approach. It splits the list into halves, recursively sorts each half, and merges the sorted halves back together.

<img width="910" src="https://github.com/SAFCSP-Team/data-structures-and-algorithms-bootcamp/assets/148945652/47e8aa7f-c559-4a9a-8342-7832283159d1" />

**Example: sorting `[5, 1, 3, 9, 4, 2, 7, 6]`**

1. **Divide**: split into `[5, 1, 3, 9]` and `[4, 2, 7, 6]`, then keep dividing until each part has one element.
2. **Sort**: each single-element part is trivially sorted: `[1, 5]`, `[3, 9]`, `[2, 4]`, `[6, 7]`.
3. **Merge**: combine pairs back together in order. Merging `[1, 5]` with `[3, 9]` gives `[1, 3, 5, 9]`, and merging `[2, 4]` with `[6, 7]` gives `[2, 4, 6, 7]`. Merging those two results produces the final sorted list: `[1, 2, 3, 4, 5, 6, 7, 9]`.

### Implementation

```java
public class MergeSort {

    public static void mergeSort(int[] arr) {
        if (arr.length <= 1) {
            return;
        }

        int mid = arr.length / 2;
        int[] left = new int[mid];
        int[] right = new int[arr.length - mid];

        // Divide the array into two halves
        System.arraycopy(arr, 0, left, 0, mid);
        System.arraycopy(arr, mid, right, 0, arr.length - mid);

        // Recursively sort the two halves
        mergeSort(left);
        mergeSort(right);

        // Merge the sorted halves back into arr
        merge(arr, left, right);
    }

    public static void merge(int[] arr, int[] left, int[] right) {
        int i = 0, j = 0, k = 0;

        while (i < left.length && j < right.length) {
            if (left[i] <= right[j]) {
                arr[k] = left[i];
                i++;
            } else {
                arr[k] = right[j];
                j++;
            }
            k++;
        }

        while (i < left.length) {
            arr[k] = left[i];
            i++;
            k++;
        }

        while (j < right.length) {
            arr[k] = right[j];
            j++;
            k++;
        }
    }

    public static void printArray(int[] arr) {
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] numbers = {9, 2, 5, 1, 7, 4};

        System.out.print("Unsorted numbers: ");
        printArray(numbers);

        mergeSort(numbers);

        System.out.print("Sorted numbers: ");
        printArray(numbers);
    }
}
```

**Output:**
```
Unsorted numbers: 9 2 5 1 7 4 
Sorted numbers: 1 2 4 5 7 9 
```

>[!NOTE]
> `System.arraycopy()` copies elements from a source array to a destination array. Here, `left` holds indices `0` to `mid - 1`, and `right` holds the rest.

## Quick Sort

Quick sort also follows **divide-and-conquer**. It picks a `pivot` element, partitions the array so smaller elements go left and larger elements go right, then recursively sorts each partition.

<img width="910" src="https://github.com/SAFCSP-Team/data-structures-and-algorithms-bootcamp/assets/148945652/2743bd5e-b721-4e6a-af71-b8adb883d835" />

**Example: sorting names `[Sarah, Mark, Emma, John, Alex, Lisa]` alphabetically**

1. Pick `Emma` as the pivot.
2. Partition: names before `Emma` go left → `[Alex]`, names after go right → `[Sarah, Mark, John, Lisa]`.
3. Recursively sort each partition the same way until every sub-list has one element.
4. Combine: `[Alex, Emma, John, Lisa, Mark, Sarah]`.

### Implementation

```java
public class QuickSort {

    static void swap(int[] arr, int i, int j) {
        int temp = arr[i];
        arr[i] = arr[j];
        arr[j] = temp;
    }

    // Partitions the array and returns the pivot's final index
    static int partition(int[] arr, int low, int high) {
        int pivot = arr[high]; // last element as the pivot
        int i = low - 1;

        for (int j = low; j < high; j++) {
            if (arr[j] <= pivot) {
                i++;
                swap(arr, i, j);
            }
        }

        swap(arr, i + 1, high);
        return i + 1;
    }

    static void quickSort(int[] arr, int low, int high) {
        if (low < high) {
            int pivotIndex = partition(arr, low, high);

            quickSort(arr, low, pivotIndex - 1);
            quickSort(arr, pivotIndex + 1, high);
        }
    }

    static void printArray(int[] arr) {
        for (int num : arr) {
            System.out.print(num + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] arr = {7, 2, 1, 6, 8, 5, 3, 4};

        System.out.print("Original array: ");
        printArray(arr);

        quickSort(arr, 0, arr.length - 1);

        System.out.print("Sorted array: ");
        printArray(arr);
    }
}
```

**Output:**
```
Original array: 7 2 1 6 8 5 3 4 
Sorted array: 1 2 3 4 5 6 7 8 
```
